## Деплой (GitHub + Vercel + база)

- Репозиторият в GitHub е източникът на кода; **Vercel** деплой към **https://colortrack.vercel.app**.
- **Базата (Neon и др.)** е вързана към проекта във Vercel: `DATABASE_URL` и останалите секрети се задават само в **Environment Variables** на Vercel (или локално в `backend/.env` — файлът е в `.gitignore` и **не се комитира**).
- За списък на променливите виж `backend/.env.example` и корен `.env.example`. Миграциите се прилагат при старт на API (`schemaEnsure`).
